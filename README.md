# UFC Fight Prediction

Сервис для прогнозирования исходов боёв UFC с объяснением причин победы. Пользователь вводит двух бойцов, а система выдаёт вероятность победы бойца, и ключевые факторы, повлиявшие на результат.

---

## **Стек проекта**

- **Backend:** FastAPI (Flask)
- **Модель:** Предобученная ML-модель
- **Базы данных:**
  - **PostgreSQL** – основное хранилище бойцов, боёв, турниров 
  - **Redis** – кэш результатов прогнозов для быстрого доступа ( TTL )
  - **(Опционально) MongoDB** – хранение истории прогнозов и метрик 
- **RabbitMQ** – очередь задач для асинхронного прогноза боёв

---

## **Архитектура**
Пример:
	1.	Пользователь запрашивает прогноз.
	2.	Backend проверяет Redis:
	•	Если есть → возвращает мгновенно.
	•	Если нет → кладёт задачу в RabbitMQ.
	3.	Worker обрабатывает:
	•	Берёт статистику бойцов из PostgreSQL.
	•	Прогоняет модель → получает победителя, вероятность, причины.
	•	Сохраняет результат в MongoDB (история) и Redis (кэш).
	4.	Backend отдаёт прогноз пользователю.
